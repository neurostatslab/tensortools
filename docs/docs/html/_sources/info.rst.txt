Change Log
----------

Release 0.2 - (August 2018)
^^^^^^^^^^^^^^^^^^^^^^^^^^^
* We gave the code a serious overhaul. For legacy code checkout commit ``e08f1f47379f814b191f2137cc1cf69f0b444be7``.
* Basic algorithms and functionality for CP tensor decomposition are available and stable. In the next release, we hope to implement randomized algorithms, basic regularization, and cross-validation.
